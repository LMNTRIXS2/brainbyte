CREATIVE DESIGNING

Game designing

Brain-Byte is a game composed of different mazes with different interesting features so that we can have a single-player mode, where the player tries to escape the maze on his own and a multi-player game, where players are allowed to help each other out in order to escape the mazes having different difficulties. The game will involve a three-dimensional world that provides an eerie atmosphere.
